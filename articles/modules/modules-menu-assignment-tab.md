## Descrição

A aba *Módulo: Atribuição de Menu* é usada, com pequenas variações, em todos os formulários de edição de módulos do site para controlar quais páginas exibem o módulo selecionado.

## Campos de Formulário

### Guia de Atribuição de Menu

* **Atribuição de Módulos** Selecione uma das quatro opções disponíveis.
    * *Em todas as páginas* O módulo será exibido em todas as páginas do site se estiver atribuído a uma **Posição** do template.
    * *Nenhuma Página* O módulo não será exibido em nenhuma página quando atribuído a uma Posição do template. Ele será exibido se estiver embutido em um artigo.
    * *Apenas nas páginas selecionadas* Uma lista de todos os itens de menu do site aparece para permitir a seleção das páginas em que o módulo deve aparecer.
    * *Em todas as páginas exceto as selecionadas* Uma lista de todos os itens de menu do site aparece para permitir a seleção das páginas em que o módulo não deve aparecer.
